## [2026-05-12] - Nuxt runtime pixel config compatibility

### Fixed

- **Issue**: Nuxt runtime plugin switched to `runtimeConfig.public.mmpPixelId` without explicit backward compatibility coverage for legacy config usage.
  - **Root cause**: Legacy `runtimeConfig.mmpApiKey` mapping was removed during client-side config fix.
  - **Solution**: Added fallback to `runtimeConfig.mmpApiKey` when `public.mmpPixelId` is not set and documented it as deprecated.
  - **Files modified**: `src/runtime/plugin.ts`, `tests/unit/plugin.test.ts`, `src/nuxt.ts`, `README.md`
  - **Status**: ✅ Fixed

### Added

- Runtime plugin tests validating:
  - `runtimeConfig.public.mmpPixelId` maps to plugin `pixelId`
  - Legacy `runtimeConfig.mmpApiKey` fallback behavior
